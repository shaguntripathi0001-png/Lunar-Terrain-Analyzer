# Lunar Terrain Analyzer
The Lunar Terrain Analyzer is an AI-powered
application designed to analyze lunar images.

The system can detect craters, generate
terrain relief information and identify
potential safe landing zones for lunar missions.

## Features

- Crater Detection
- Terrain Relief Analysis
- Safe Landing Zone Identification
- Interactive GUI
- Telemetry Dashboard
- Image Upload Support
- Zoom and Pan Functionality

- ## Technologies

- Python
- OpenCV
- YOLO
- NumPy
- Tkinter
- PIL

- ## Future Improvements
